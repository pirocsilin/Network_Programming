    Изучение дисциплины “Сетевое программирование” ставит своей целью подготовку специалистов для производственной деятельности в области разработки сетевых приложений и технической эксплуатации инфокоммуникационных систем и для проведения исследований в области разработки перспективных систем передачи.

    В дисциплине изучаются основные принципы сетевого программирования при разработке новых сетевых приложений, что даёт возможность узнать как работают сетевые компоненты операционных систем. Рассматривается деление таких приложений на клиентскую и серверную части; представлены примеры программ клиентов и серверов. Большое внимание уделяется архитектуре TCP/IP сетей и вопросам реализации функций сетевого программирования для API сокетов (socket, bind, connect и т.д.).

    В процессе изучения данной дисциплины Вам необходимо:


    • Изучить конспект лекций
    • Выполнить курсовую работу
    • Выполнить 3 лабораторных работы
    • Сдать экзамен

    Внимательно прочтите следующие рекомендации:

    1. Курсовая работа выполняется по индивидуальному варианту в соответствие с методическими указаниями по ее выполнению.
    2. При выполнении курсовой работы рекомендуется пользоваться литературой.
    3. Номер Вашего варианта определяется последней цифрой пароля. Курсовая работа и отчеты по лабораторным работам оформляются в доступном редакторе.
    4. Выполненные задания должны быть высланы со страниц нашего сайта. С порядком отправки выполненных работ можно ознакомиться в инструкции по отправке, доступной по ссылке.
    5. После проверки курсовой работы и отчетов преподавателем Вы получите рецензии с оценками и возможными замечаниями, которые следует устранить. Если из-за ошибок какая-либо из работ окажется не зачтена, то ошибки необходимо устранить. В этом случае после доработки Вы должны повторно выслать ее на проверку.
    6. Вы будете допущены к экзамену только после получения положительной оценки по курсовой работе и зачета по лабораторным работам..

    Получить консультацию преподавателя можно по контактным данным (см. раздел "Консультации ").
